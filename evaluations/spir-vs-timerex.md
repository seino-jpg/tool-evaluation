# Spir vs Timerex 料金比較

調査日: 2026-03-27

## 個人プラン

| 項目 | Spir Free | Timerex フリー |
|---|---|---|
| 料金 | **無料** | **無料** |
| 利用人数 | 2人まで | 無制限 |
| 空き時間リンク | 3つまで | 基本機能 |
| CC登録 | 不要 | 不要 |

## 有料プラン（1ライセンスあたり）

| 項目 | Spir Team | Timerex ベーシック | Timerex プレミアム |
|---|---|---|---|
| 月払い | **1,500円** | **900円** | **1,500円** |
| 年払い | **1,200円** | **750円** | **1,250円** |
| 最低人数 | 1人〜 | 1人〜 | 1人〜 |
| 大人数 | 要問い合わせ | 40名以上はエンタープライズ | 同左 |

## 機能の差異

| 機能 | Spir Team | Timerex ベーシック | Timerex プレミアム |
|---|---|---|---|
| 会議室自動予約 | ✅ | ✅ | ✅ |
| Webサイト埋め込み | ✅ | - | - |
| Webhook連携 | ✅ | - | - |
| 質問フォーム | ✅ | - | ✅ |
| Salesforce/HubSpot連携 | - | - | ✅ |
| 参加者自動決定 | - | ✅ | ✅ |

## Webhookの強み（Spir）

Timerexの「Salesforce/HubSpot連携」は特定SaaSとの固定連携だが、SpirのWebhookは**任意のシステムに自由に繋げられる**。

| ユースケース | 具体例 |
|---|---|
| CRM自動更新 | 商談確定 → Salesforceの案件ステータスを自動更新 |
| Slack通知 | 顧客がURLから予約 → 担当者にSlack通知 |
| 社内DB連携 | 予約確定 → スプレッドシート・Notionに自動記録 |
| 業務フロー起動 | 打ち合わせ確定 → 事前アンケート送付・議事録テンプレ作成 |
| リマインダー連携 | Zapier/Make経由でSMSやメール自動送信 |

## まとめ

| 観点 | 推奨 |
|---|---|
| コスト重視 | Timerex（年払い750円〜、Spirより約40%安い） |
| Webhook・埋め込みが必要 | Spir |
| CRM連携（Salesforce/HubSpot） | Timerex プレミアム |
| エンジニア・自動化チーム | Spir（Webhookの自由度が高い） |
